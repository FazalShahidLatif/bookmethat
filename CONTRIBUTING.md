# Contributing to bookmethat.com

Thank you for your interest in contributing!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/bookmethat.git`
3. Install dependencies: `npm install`
4. Copy `.env.example` files and configure
5. Run development servers: `npm run dev:all`

## Branch Naming

- `feature/` - New features
- `fix/` - Bug fixes
- `docs/` - Documentation changes
- `refactor/` - Code refactoring
- `test/` - Adding tests

## Commit Messages

Follow conventional commits:
- `feat:` - New feature
- `fix:` - Bug fix
- `docs:` - Documentation
- `style:` - Formatting
- `refactor:` - Code refactoring
- `test:` - Tests
- `chore:` - Maintenance

Example: `feat: add property search filters`

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Add tests
4. Update documentation
5. Run `npm run lint` and `npm test`
6. Push and create a PR
7. Wait for review

## Code Style

- Use TypeScript
- Follow ESLint rules
- Use Prettier for formatting
- Write tests for new features
- Add JSDoc comments for complex functions

## Questions?

Open an issue or reach out to the maintainers.
